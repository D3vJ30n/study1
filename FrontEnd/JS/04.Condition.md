## 조건문
조건문은 프로그래밍에서 매우 중요한 개념 중 하나다. 조건문을 통해 프로그램의 흐름을 제어하고 원하는 로직을 구현할 수 있다.

## 1. If문
if문은 주어진 조건식의 평가 결과에 따라 코드 블록의 실행 여부를 결정다. 조건식은 boolean 값으로 평가되어야 하며, 참(true)인 경우에만 해당 코드 블록이 실행된다. if문의 기본 구조는 다음과 같다.

```js
if (조건식) {
  // 조건식이 참일 때 실행되는 코드 블록
} else {
  // 조건식이 거짓일 때 실행되는 코드 블록
}
```
else 블록은 생략 가능하며, 조건식이 거짓일 때 실행되는 코드 블록이다.
if문은 중첩하여 사용할 수 있습니다. 중첩된 if문을 사용하면 더 세부적인 조건을 체크할 수 있다.
<br>
<br>예를 들어
```js
if (조건식1) {
  // 조건식1이 참일 때 실행되는 코드 블록
  if (조건식2) {
    // 조건식1과 조건식2가 모두 참일 때 실행되는 코드 블록
  }
} else {
  // 조건식1이 거짓일 때 실행되는 코드 블록
}
```
또한, else if를 사용하여 여러 개의 조건을 체크할 수 있다. else if는 이전 조건식이 거짓일 때 다음 조건식을 평가한다.
```js
if (조건식1) {
  // 조건식1이 참일 때 실행되는 코드 블록
} else if (조건식2) {
  // 조건식1이 거짓이고 조건식2가 참일 때 실행되는 코드 블록
} else {
  // 조건식1과 조건식2가 모두 거짓일 때 실행되는 코드 블록
}
```
간단한 if-else문은 삼항 연산자로 대체할 수 있다. 삼항 연산자를 사용하면 코드를 더 간결하게 작성할 수 있다.

```js
조건식 ? 조건식이_참일_때_실행되는_표현식 : 조건식이_거짓일_때_실행되는_표현식
```

## 2. switch문
switch문은 특정 값에 따라 여러 개의 case로 분기하여 코드를 실행할 수 있다. switch문의 기본 구조는 다음과 같다.
```js
switch (표현식) {
  case 값1:
    // 표현식이 값1과 일치할 때 실행되는 코드 블록
    break;
  case 값2:
    // 표현식이 값2와 일치할 때 실행되는 코드 블록
    break;
  // ...
  default:
    // 표현식이 위의 어떤 case와도 일치하지 않을 때 실행되는 코드 블록
}
```
switch문에서는 표현식의 값과 일치하는 case의 코드 블록이 실행된다. 
각 case 블록 끝에는 break 문을 사용하여 switch문을 종료해야 한다. 
break 문이 없으면 일치하는 case 이후의 모든 case 블록이 실행되므로 주의해야 한다.
<br>
<br>default 블록은 표현식이 어떤 case와도 일치하지 않을 때 실행되는 코드 블록이다. default 블록은 생략 가능하다.
<br>switch문은 if-else문과 비슷한 역할을 하지만, 특정 값에 따라 분기할 때 더 가독성이 좋고 효율적이다.
<br>
<br>조건문을 적절히 활용하면 프로그램의 흐름을 제어하고 원하는 로직을 구현할 수 있다. 코드의 가독성과 효율성을 고려하여 if문과 switch문을 선택하여 사용하면 좋을 것 같다.
